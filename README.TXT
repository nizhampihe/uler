DESCRIPTION
===========
    Program Game Console Snake
  Sederhana

PREREQUISITES
=============
  - MinGW-W64 Compiler

INSTALATION
===========
  > gcc -o ULER.EXE -c ULER.C

GAMEPLAY
========

  HOW TO PLAY
  ===========
    - Gunakan tombol panah
      untuk mengubah arah
      gerak ular
    - Ambil buah untuk
      mendapatkan score
    - Semakin banyak buah
      yang diambil tubuh
      ular semakin panjang
    - Jangan sampai menabrak
      tembok
    - Jangan sampai memakan
      diri sendiri

  CONTROL
  =======
    - Panah : Jalan
    - P     : Pause
    - Esc   : Keluar

  FEATURES
  ========
    - ASCII Block Graphic
    - Pause Button
    - Play Again
    - High Score
